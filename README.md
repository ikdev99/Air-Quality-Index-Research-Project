# Air Quality Index Research Project

## Overview
This research project focuses on analyzing and predicting Air Quality Index (AQI) using machine learning and data analysis techniques.

## Objectives
- Analyze air pollution data
- Predict AQI levels
- Compare machine learning models
- Visualize environmental data

## Features
- Data preprocessing
- AQI prediction
- Model comparison
- Data visualization
- Performance evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models
- Logistic Regression
- Random Forest
- SVM
- XGBoost
- MLP
